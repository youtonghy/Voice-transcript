# Voice-Translate-Transcribe 王译转

使用Python+Electron开发的基于OpenAI transcribe模型进行转写,翻译的软件.

## 使用

请从Release中下载最新版本使用,目前只编译了Windows版本,适用于MacOS和Linux的已加入计划.

## 开发

请直接使用,可参考具体文档

```
npm run dev
```
